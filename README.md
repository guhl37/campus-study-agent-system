# 🏫 高校大学生课程助学备考多Agent系统
专为大学生打造的一站式课程学习、期末备考、论文作业辅助系统，基于多Agent协同与大模型长链推理，实现个性化学习全流程闭环。

## ✨ 核心功能
针对大学生学习的核心痛点，打造5个核心Agent协同工作，覆盖学习全流程：
1. **课程知识拆解Agent**：解析课件、考试大纲、历年真题，生成三级考点知识图谱，精准定位高频考点
2. **学情诊断Agent**：分析错题、作业情况，长链推理定位知识盲区与错误根源，生成精准学情画像
3. **学习路径生成Agent**：结合考点权重与学情画像，生成分阶段、可落地的个性化备考学习计划
4. **习题与复盘Agent**：针对薄弱知识点生成专属习题，贴合课程考点生成完整模拟考试卷，附带解析与评分标准
5. **论文作业辅助Agent**：拆解作业要求，生成规范写作大纲，完成论文格式核对与修正，适配课程论文/毕业论文等多种场景

## 🛠️ 技术栈
- **核心语言**：Python 3.10+
- **Agent框架**：LangChain（多Agent协同、长链推理实现）
- **大模型支持**：兼容OpenAI格式的所有大模型（智谱AI GLM、DeepSeek、OpenAI GPT等）
- **Web可视化界面**：Gradio（零前端代码，一键启动，快速演示）
- **向量知识库**：FAISS（本地轻量化向量存储，无需额外部署数据库）
- **文档解析**：支持PDF、DOCX、TXT格式的课件、真题、文档解析
